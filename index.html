Projects

Checkout some exciting projects that I've recently worked on!

Power Quencher - Virtual Menu Ordering System

Power Quencher is a Java application, a personal project that I had worked on. It allows customers to make selections of their orders and simultaneously view their receipt. I have applied User interface design to meet general specifications of clients. The application is a form of virtual menu for companies to present their service and products online.

Chatle - Play Wordle and Chat

Inspired by the popular word game, Wordle - Chatle is a program that allows you to play Wordle while chatting with friends and other online users. The program utilizes sockets to design a client-server model. The purpose of this project is to exemplify a Network OS, where the server contains the main program file and required resources to run the application.

Handwritten Digit Reader - Utilizes Machine Learning

Handwritten Digit Reader is a python program that utilizes AI to accurately predict a computable value from an image of a handwritten digit. The program is implemented using Machine Learning algorithms to read and train with MNIST data. It utilizes various python libraries to perform supervised and reinforced learning: tensorflow, scikit-learn, pickle, numpy, matplotlib, scipy.
